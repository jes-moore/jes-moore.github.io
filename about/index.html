Some Advice For Myself

  • Excel in a career that mixes opportunity with something you love doing.
  • Set deadlines. Even if you don't think you need them, you need them, so set them.
  • Don't be afraid of change.
  • Strive to be a better person. Be a better person, be kind, be giving.
